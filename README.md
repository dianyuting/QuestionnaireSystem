# 项目介绍
* 本项目是一个简易的消费者调研统计系统，是为了便于商家了解用户的消费习惯和市场需求情况。<br>
主要由登录、问卷列表、题库管理、员工管理、账号安全等部分组成。
* 该项目采用SSM框架
* 开发工具：eclipse+VSCode+Navicat<br>
开发环境：jdk1.8+mysql5+tomcat8.5
